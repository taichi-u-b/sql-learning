# SQL Learning

## 目的
データ基盤SEとして必要なSQLとPostgreSQLの基礎を学習し、学習内容を記録する。

## 学習内容
- SQLの基本構文
- PostgreSQLの基本操作
- SELECT / WHERE / ORDER BY
- GROUP BY / HAVING
- JOIN
- サブクエリ
- データベース設計の基礎

## 学習ルール
- 各単元ごとに `memo.md` と `practice.sql` を作成する
- `memo.md` には概念・構文・注意点をまとめる
- `practice.sql` には実際に書いたSQLを残す
- 間違えた内容は `mistakes.md` に記録する

## 使用環境
- PostgreSQL
- pgAdmin
- psql
